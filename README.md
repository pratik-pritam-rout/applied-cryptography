# Modern Cryptography

This repository contains my C++ implementations of modern cryptographic algorithms, public-key cryptosystems, primality testing methods, and supporting number theory concepts.

The primary purpose of this repository is to help me revise and strengthen my understanding of modern cryptography through hands-on implementation of the underlying algorithms and mathematical concepts.

## Topics Covered

### Public-Key Cryptography

* RSA Cryptosystem
* Diffie-Hellman Key Exchange
* ElGamal Cryptosystem
* Rabin Cryptosystem
* Paillier Homomorphic Encryption
* Elliptic Curve Cryptography (ECC)

### Number Theory

* Greatest Common Divisor (GCD)
* Extended Euclidean Algorithm
* Modular Inverse
* Legendre Symbol
* Jacobi Symbol
* Quadratic Residues

### Primality Testing

* Fermat's Primality Test
* Miller-Rabin Primality Test
* Solovay-Strassen Primality Test

### Security Demonstrations

* Man-in-the-Middle Attack Simulation

## Purpose

This repository is intended for educational purposes and personal revision. The implementations focus on understanding the mathematical foundations and working principles of modern cryptographic systems rather than providing production-ready cryptographic software.
